MessageEnvelope

MessageEnvelope class

Allows adding additional information to messages. A trace id, message id, and a message type are added to the data being sent/received. Additionally, a MessageEnvelope can reference a lock token.

Side note: a MessageEnvelope's message is stored as string, so strings are converted using utf8 conversions.

Creates a new MessageEnvelope, which adds a trace id, message id, and a type to the data being sent/received.
